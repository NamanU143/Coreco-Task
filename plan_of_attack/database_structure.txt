Tables for Asset Management System

Users
id (Primary Key, Auto Increment)
name (VARCHAR)
email (VARCHAR, Unique)
password (VARCHAR)
role (ENUM: Admin, User)
status (Active, Inactive)


Assets
id (Primary Key, Auto Increment)
name (VARCHAR)
description (TEXT)
asset_type_id (Foreign Key → Asset_Types)
status (Available, Assigned, Under Maintenance)
purchase_date (DATE)
assigned_to (Foreign Key → Users, Nullable)


Asset_Types
id (Primary Key, Auto Increment)
type_name (VARCHAR, Unique)
description (TEXT)


Transactions
id (Primary Key, Auto Increment)
asset_id (Foreign Key → Assets)
from_user (Foreign Key → Users, Nullable)
to_user (Foreign Key → Users)
transaction_date (DATETIME)
status (Pending, Approved, Rejected)


Maintenance_Records
id (Primary Key, Auto Increment)
asset_id (Foreign Key → Assets)
start_date (DATE)
end_date (DATE, Nullable)
description (TEXT)
status (In Progress, Completed)


Requests
id (Primary Key, Auto Increment)
user_id (Foreign Key → Users)
asset_id (Foreign Key → Assets)
request_type (Assign, Return, Repair)
status (Pending, Approved, Rejected)
created_at (DATETIME)


Logs
id (Primary Key, Auto Increment)
user_id (Foreign Key → Users)
action (VARCHAR)
timestamp (DATETIME)
details (TEXT)
